## Image Guidelines

### Style Requirements
- **Color Palette**: Images should complement the Black + Gold + Salmon color scheme
- **Mood**: Elegant, premium, authentic Japanese cuisine
- **Lighting**: Professional food photography lighting
- **Composition**: Clean, focused, restaurant-quality presentation

### Technical Requirements
- **Format**: JPEG for photos, PNG for graphics with transparency
- **Optimization**: Compress for web (aim for <200KB per image)
- **Responsive**: Provide @2x versions for retina displays where appropriate
- **Alt Text**: All images should have descriptive alt text for accessibility

- All placeholder images are currently using generic stock photos or existing chef images
